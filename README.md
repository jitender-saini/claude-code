# claude-code

Utilities and enhancements for [Claude Code](https://claude.ai/claude-code).

## Status Line

A rich terminal status line that displays model, context usage, tokens, git branch, cost, and more.

**One-line install:**

```bash
curl -fsSL https://raw.githubusercontent.com/jitender-saini/claude-code/main/statusline/install.sh | bash
```

See [statusline/README.md](statusline/README.md) for details, manual install instructions, and customization tips.
